# Security Policy

## Declarative Intention
CacheFlow is an open-source utility designed exclusively for legitimate cache management and storage optimization on Android devices with the user's full consent.

## Reporting a Vulnerability
The CacheFlow team takes security seriously. If you discover a security vulnerability within this project, please follow these steps:

1. **Do not disclose the vulnerability publicly.**
2. Send a detailed report to **security@sahraouilarbi.com**.
3. We will acknowledge receipt of your report within 48 hours.
4. We aim to analyze and confirm the vulnerability within 7 days and provide a remediation plan.

## Prohibited Uses
Any use of this software, its source code, or any derivative work for the purpose of:
- Unauthorized surveillance or data exfiltration.
- Bypassing user consent or security controls.
- Developing malware, spyware, or ransomware.
- Any other malicious activity.

... is strictly prohibited and violates the spirit of the GPL-v3 license under which this project is distributed.

## Responsible Disclosure
We encourage the community to contribute to the security of CacheFlow. We support a responsible disclosure process and will provide credit to researchers who help us improve the safety of our users.
